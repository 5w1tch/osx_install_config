### macOS

reboot to recovery (cmd +R)

	set secure boot in recovery (only on macs with a t2 processor)
	- only allow signed sources
	- do not allow boot from usb
	- set recovery password

	set SIP in recovery
	csrutil enable --without fs --without debug

	reboot

open "mail" and keep open in background for indexing

virus scanner
	check if virus scan is autostarted (symbol "B" in menu bar, eventually in bartender)
	virus scan (menu bar) - open main window - "user-defined scan" - complete disk (macintosh_hd)

print filevault_key

delete restore files on desktop and empty trash

add card(s) to apple wallet (only possible with sip fully enabled)

enable touch id

clear list of old devices in icloud acccount
